Convert XLSX ke TXT

Fitur konversi format file

Mengubah file Excel (.xlsx) menjadi file teks (.txt) dengan format sederhana: satu baris berisi satu nomor. File Excel diasumsikan memiliki 1 kolom berisi data nomor yang akan dibaca.

  • Hanya menerima file berekstensi .xlsx
  • Membaca kolom pertama dari sheet pertama
  • Setiap sel yang berisi data = 1 baris teks
  • Sel kosong akan dilewati otomatis
  • Output file .txt langsung bisa di-download
  • Semua proses di perangkat, tanpa kirim ke server

Pecah File

Fitur split data otomatis

Memecah data dari file .xlsx atau .txt menjadi beberapa file teks kecil. Setiap file berisi maksimal 1000 baris data. Semua file hasil digabung menjadi satu arsip ZIP.

  • Menerima file .xlsx dan .txt
  • .xlsx dibaca pakai library XLSX di browser
  • .txt dibaca per baris sebagai data teks
  • Setiap file output maksimal 1000 baris
  • Sisa data tetap dibuatkan file terakhir
  • 10.000 data menghasilkan 10 file (masing-masing 1000 baris)
  • 5.500 data menghasilkan 6 file (5 x 1000 + 1 x 500)
  • Semua file dikemas jadi satu .zip via JSZip
  • Nama file otomatis: nama_1.txt, nama_2.txt, dst.

Contoh: 2.350 data menghasilkan 3 file (2 file isi 1000 baris + 1 file isi 350 baris), semua dalam satu file ZIP.

Keamanan Data

Privasi terjaga sepenuhnya

Aplikasi ini berjalan sepenuhnya di browser perangkatmu. Tidak ada data yang dikirim ke server manapun. Setelah proses selesai, file yang di-upload tidak disimpan dan akan hilang saat halaman ditutup.
